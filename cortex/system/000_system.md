# System

Parte técnica do CortexCore.

## Objetivo
Documentar a infraestrutura: onde o sistema vive, como é guardado, protegido e copiado.

## O que vai guardar aqui
- [[cortex/system/github|github]] — repositório privado
- [[cortex/system/server|server]] — servidor / VPS
- [[cortex/system/security|security]] — segurança
- [[cortex/system/backups|backups]] — cópias de segurança
- [[cortex/system/gitignore|gitignore]] — o que não vai para o Git

## Princípio central
- Código, estrutura e templates **podem** ir para o GitHub privado.
- Dados sensíveis **não** devem ir para o GitHub em bruto.
- Separação clara entre o que é público/estrutural e o que é privado.

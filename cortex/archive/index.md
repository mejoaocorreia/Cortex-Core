# Cortex Archive

Arquivo do núcleo.

## Objetivo
Guardar material antigo do `cortex` que já não está ativo, mas vale a pena manter.

## Regra
Cada área deve ter o **seu próprio** archive. Este archive é apenas para coisas do núcleo. O [[archive/index|archive global]] da raiz é para material global ou sem área clara.

## O que vai guardar aqui
- versões antigas de protocolos, rotinas ou notas do núcleo

## Como o CortexCore vai usar
Última camada de leitura (ver [[cortex/protocols/context_ladder|context_ladder]]): só consultado quando relevante.

# Cortex Assets (placeholder)

Assets do núcleo CortexCore.

## Objetivo
Guardar ficheiros e anexos que pertencem ao **núcleo** do CortexCore — interface e sistema. **Não** é uma pasta global para todos os anexos do Vault.

## O que vai guardar aqui
- imagens/ícones da [[cortex/interface/000_interface|interface]]
- diagramas e recursos do sistema/núcleo

## O que NÃO fica aqui
Ficheiros específicos de uma área ou projeto devem viver **junto dessa área/projeto**:
- ficheiros do website → `professional/previmed/website/`
- ficheiros da Previmed → `professional/previmed/`
- ficheiros do carro → `car/`
- ficheiros de saúde → `health/`

## Como o CortexCore vai usar
Ponto único para os recursos visuais do núcleo. Para regras gerais de anexos, ver [[cortex/protocols/assets|protocols/assets]].

## Aviso de privacidade
Anexos podem conter **dados sensíveis**. Seguir [[cortex/protocols/privacy|privacy]]: nada de dados sensíveis em bruto no GitHub.

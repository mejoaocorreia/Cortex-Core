# Daily Archive

Arquivo de dias passados.

## Objetivo
Guardar os `today.md` antigos, organizados por data.

## O que vai guardar aqui
- estrutura sugerida: `ano/mês/data.md` (ex.: `2026/05/2026-05-23.md`)

## Como o CortexCore vai usar
O [[cortex/routines/daily_rollover|daily_rollover]] move o `today` para aqui quando muda o dia. Não é preciso criar os ficheiros datados já — são criados ao longo do tempo.

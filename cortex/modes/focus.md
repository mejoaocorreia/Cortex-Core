# Focus Mode

Modo de concentração numa tarefa.

## Objetivo
Reduzir tudo a uma só coisa: a tarefa atual.

## O que vai guardar aqui
- definição do que aparece (e do muito que desaparece) em foco máximo

## Como o CortexCore vai usar
Mostra apenas o item ativo da [[cortex/queue/priority|priority]] / [[cortex/daily/focus|focus]] e esconde o resto. Lente para concentração profunda.

## Ideias discutidas
- Foco é escolher pouco: uma tarefa de cada vez.
- Diferente de [[cortex/modes/build|Build Mode]]: focus é uma tarefa; build é uma sessão de criação.

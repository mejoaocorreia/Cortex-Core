# Boot Mode

Modo de arranque do sistema.

## Objetivo
O que o CortexCore mostra ao "ligar" — o estado essencial para começar.

## O que vai guardar aqui
- definição do que aparece no arranque (painel mínimo)
- atalhos para o que importa primeiro

## Como o CortexCore vai usar
Mostra o painel [[000_boot|boot]], o [[cortex/status/overall|overall status]] e o que precisa de atenção imediata. É a primeira lente ao abrir o Vault.

## Relação
- Pode ser acionado pela [[cortex/routines/boot|boot routine]].

## Ideias discutidas
- Arranque deve ser leve: orientar, não sobrecarregar.

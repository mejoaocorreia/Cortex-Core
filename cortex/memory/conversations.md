# Conversations

Notas de conversas relevantes.

## Objetivo
Guardar resumos de conversas que originaram decisões ou direções, sem reproduzir dados sensíveis.

## Como o CortexCore vai usar
Quando uma conversa (com pessoas ou com agentes) gera algo importante, fica aqui um resumo com a conclusão — não a transcrição completa.

## Notas
> Preencher ao longo do tempo. Formato sugerido:
> - **Data:** AAAA-MM-DD
> - **Sobre:** ...
> - **Conclusão:** ...

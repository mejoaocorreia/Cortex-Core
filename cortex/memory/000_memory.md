# Memory

Memória do próprio CortexCore. (`memory` substitui `logs`.)

## Objetivo
Guardar a evolução do sistema: decisões, ideias e histórico. É a memória de longo prazo do CortexCore sobre si mesmo.

## O que vai guardar aqui
- [[cortex/memory/decisions|decisions]] — decisões tomadas e o seu porquê
- [[cortex/memory/ideas|ideas]] — ideias por explorar
- [[cortex/memory/changelog|changelog]] — registo de alterações ao sistema
- [[cortex/memory/conversations|conversations]] — notas de conversas relevantes

## Como o CortexCore vai usar
Antes de repetir uma decisão ou debate, consulta-se a memória. Novas decisões e ideias são registadas aqui para não se perderem.

## Ideias discutidas
- "memory" descreve melhor o propósito do que "logs" (não é só registo técnico).

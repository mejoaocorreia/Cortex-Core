# Decisions

Registo de decisões do CortexCore.

## Objetivo
Guardar decisões importantes e o raciocínio por trás delas, para não se repetirem debates já fechados.

## Como o CortexCore vai usar
Cada decisão relevante é anotada com data, contexto e motivo. Usar o [[cortex/templates/template_decision|template de decisão]].

## Decisões registadas
> Preencher ao longo do tempo. Exemplo de formato (sem dados reais):
> - **Data:** AAAA-MM-DD
> - **Decisão:** ...
> - **Porquê:** ...
> - **Alternativas consideradas:** ...

# Versão 0.00.0

Primeira versão do CortexCore.

## Objetivo
Criar a base de pastas e ficheiros explicativos do Vault. Definir a estrutura mental e a nomenclatura, sem ainda inserir dados reais.

## Estado
Estruturação inicial.

## Decisões já tomadas
- O Vault inteiro representa o CortexCore; não há pasta `cortexcore`.
- O núcleo operativo chama-se `cortex/`.
- `self` substitui `identity`.
- `memory` substitui `logs`.
- `routines` substitui `automations`.
- Fila de atenção chama-se `queue`.
- A pasta de pessoas/relações chama-se `social` (decisão fechada; não `circle`/`people`/`relationships`).
- Agentes não vivem todos numa pasta global: cada área pode ter os seus.
- Cada área tem o seu próprio `archive`; o archive global é só para coisas sem área clara.
- Nomes em minúsculas, com underscore quando necessário, e "agents" no plural.

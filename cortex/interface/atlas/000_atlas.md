# Atlas

Camada de mapas navegáveis do CortexCore.

## Objetivo
O Atlas é a camada de **mapas navegáveis** estilo Obsidian. Serve para criar mapas por área e ajudar a navegar visualmente o Vault. **Não** é um dashboard funcional nem uma automação.

## Como o CortexCore vai usar
Usa **links internos**, **Local Graph** e, futuramente, a [[cortex/interface/000_interface|interface]] própria para navegar. Complementa as [[cortex/interface/views|views]] e os [[cortex/interface/canvas/_temp|canvas]]. O ficheiro-ponteiro antigo é [[cortex/interface/atlas|atlas.md]].

## Ideias discutidas

- Mais áreas (ex.: learning) podem ganhar mapa quando fizer sentido.

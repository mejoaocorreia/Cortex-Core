# Operator — Preferences

Preferências do Operator.

## Objetivo
Registar como o Operator gosta de trabalhar, comunicar e organizar — para o sistema se adaptar a ele.

## O que vai guardar aqui
- como gosta de trabalhar
- como gosta de comunicar
- como gosta de organizar
- preferências de resposta dos agentes (tom, comprimento, detalhe)
- preferências de interface / sistema

## Como o CortexCore vai usar
Os agentes leem estas preferências para personalizar respostas e o comportamento do sistema. Liga-se ao [[cortex/self/personality|personality]] e aos [[cortex/self/operating_principles|operating_principles]].

## Ideias discutidas
- Preferências são do **Operator** (a pessoa); a personalidade é do **sistema** ([[cortex/self/000_self|self]]). As duas devem encaixar.
